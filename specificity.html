Specificity:

the means by which browsers decide which CSS property values are the most relevant to an element and, therefore, will be applied. Specificity is based on the matching rules which are composed of different sorts of CSS selectors.


Specificity is how Chrome decides what property values (colors, sizes, etc.) to apply to a webpage. It is done by matching the rules based on the selectors used in html and the stylesheet.

The selector types are:

  • Type selectors (e.g., h1) and pseudo-elements (e.g., ::before).
  • Class selectors (e.g., .example), attributes selectors (e.g., [type="radio"]) and pseudo-classes (e.g., :hover).
  • ID selectors (e.g., #example).s based on the matching rules which are composed of different sorts of CSS selectors.
